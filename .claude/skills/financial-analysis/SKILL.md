# 金融分析 Skill

## 概述
针对中国场内基金（ETF）的分析方法论，用于筛选优质基金并生成投资建议。

## 基金评估维度

### 1. 收益指标
- **近1年涨幅**: 核心排名指标
- **近3月/6月涨幅**: 中短期趋势
- **年化收益率**: 长期表现
- **最大回撤**: 风险控制能力

### 2. 规模与流动性
- **基金规模**: 优选 > 2亿元，避免清盘风险
- **日均成交额**: 优选 > 500万元，确保流动性
- **跟踪误差**: ETF特有，越小越好

### 3. 费率
- **管理费率**: 通常 0.15%-0.5%
- **托管费率**: 通常 0.05%-0.1%
- **买卖佣金**: 场内交易，与股票相同

## 行业分析框架

### 互联网行业
- 关注指标: 用户增长、ARPU、变现率
- 代表指数: 中证互联网指数、恒生科技指数
- 风险: 政策监管、流量见顶

### AI/人工智能行业
- 关注指标: 研发投入、算力需求、应用落地
- 代表指数: 中证人工智能指数、科创芯片指数
- 风险: 估值泡沫、技术迭代

### 电力行业
- 关注指标: 用电量增速、电价改革、电网投资
- 代表指数: 中证电力指数、电力设备指数
- 风险: 政策变动、煤价波动

### 新能源行业
- 关注指标: 装机量、渗透率、技术路线
- 代表指数: 中证新能源指数、光伏产业指数
- 风险: 产能过剩、补贴退坡

## 投资建议生成规则

### 资产配置原则（10万本金）
```
保守型 (风险厌恶):
  - 货币基金/债券: 50% (5万)
  - 行业ETF: 30% (3万)
  - 现金储备: 20% (2万)

均衡型 (适中风险):
  - 行业ETF: 50% (5万)
  - 宽基指数: 30% (3万)
  - 现金储备: 20% (2万)

进取型 (风险偏好):
  - 行业ETF: 70% (7万)
  - 宽基指数: 20% (2万)
  - 现金储备: 10% (1万)
```

### 行业配置建议
- 单一行业不超过总仓位的 30%
- 至少配置 3 个不同行业
- 高波动行业（AI）配置比例适当降低
- 防御性行业（电力）可适当提高

### 定投策略
- 推荐周定投，分散买入时点
- 单次定投金额 = 月投资额 / 4
- 设置止盈线（如 +20%）和止损线（如 -15%）

## 风险提示模板
```
⚠️ 投资有风险，入市需谨慎
- 以上建议仅供参考，不构成投资建议
- 基金过往业绩不代表未来表现
- 请根据自身风险承受能力做出决策
- 建议咨询专业理财顾问
```

## 数据更新频率
- 基金净值/价格: 每个交易日收盘后更新
- 排行榜: 每日更新
- 投资建议: 每周更新一次
